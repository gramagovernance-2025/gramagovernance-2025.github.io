# GRAMA Website

Single-page website. Open `index.html` in any browser.

| Filename | Where it appears |
|---|---|
| `jeevika-meeting.jpg` | JEEViKA section — SHG meeting (left) |
| `jeevika-partnership.jpg` | JEEViKA section — government photo (right) |

Team photos: crop to 5:6 ratio (~400×480px). Research thumbnails: square (~600×600px).

## Deploy to GitHub Pages

1. Create repo → upload `index.html`, `images/` folder, `README.md`
2. Settings → Pages → main branch → / (root) → Save
3. Live at `https://gramagovernance-2025.github.io/repo-name/`
